# AI-Native 自进化框架架构

## 核心循环

```
┌─────────────────────────────────────────────────────────┐
│                    AI-Native 自进化循环                    │
│                                                         │
│   ① 操作数据  ──→  ② 自动标注  ──→  ③ 自我训练          │
│        ↑                                    │            │
│        │                                    ↓            │
│   ⑥ 自动部署  ←──  ⑤ A/B 验证  ←──  ④ 模型更新        │
│                                                         │
│   每个循环: 数据 +1%  →  准确率 +0.5%  →  飞轮加速        │
└─────────────────────────────────────────────────────────┘
```

## 三大自进化模块

### 1. AutoSkill — 自动技能生成器
```
专家视频 ──→ MediaPipe 提取关键点轨迹
         ──→ LLM 分析: 识别操作步骤、关键动作、评分标准
         ──→ 自动生成 skill.yaml + scoring.py 骨架
         ──→ 人工审核 + 微调 → 发布
```

### 2. SelfTrain — 自我训练引擎
```
用户操作数据 ──→ 匿名化
            ──→ 与专家数据对比（自动标注差距）
            ──→ 生成训练样本
            ──→ 微调评分模型（LoRA / 全量）
            ──→ 自动评估 → 发布新模型版本
```

### 3. ABTest — 自动 A/B 验证
```
新模型 v1.1 ──→ 与 v1.0 并行评估同一批数据
            ──→ 对比准确率/用户满意度
            ──→ v1.1 胜出 → 自动切换
            ──→ v1.1 失败 → 保留 v1.0 + 继续训练
```

## 数据飞轮加速器

```
用户量    数据量    模型准确率
 100  →   1K条   →   70%
 500  →   5K条   →   80%   ← AutoSkill 开始自动生成
 2K   →   20K条  →   88%   ← SelfTrain 显著提升
 10K  →  100K条  →   94%   ← ABTest 自动优化
 50K  →  500K条  →   97%   ← 接近专家水平
```

## 技术栈

| 模块 | 核心技术 | 依赖 |
|------|---------|------|
| AutoSkill | MediaPipe + LLM + Prompt Eng | LLM API / Ollama |
| SelfTrain | PyTorch/TF Lite + LoRA | 本地 GPU 或 Colab |
| ABTest | SQLite + 统计检验 | 纯 Python |
| 数据管道 | SQLite + 联邦学习协议 | 内置 |
